# Specification

## Summary
**Goal:** Replace Arcanus with Luffy from One Piece and improve question relevance for character guessing.

**Planned changes:**
- Remove Arcanus character from the database and add Monkey D. Luffy with appropriate anime character attributes
- Review and update question bank to ensure all questions effectively help distinguish between characters
- Update answer patterns for all characters to accurately match the revised question set
- Replace Arcanus avatar with Luffy avatar in the frontend UI

**User-visible outcome:** Users will see Luffy from One Piece as a guessable character with relevant questions that effectively narrow down character options during gameplay.
